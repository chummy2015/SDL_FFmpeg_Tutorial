# SDL_FFmpeg_Tutorial
FFmpeg编码并播放视频，解码音频文件，参考最新FFmpeg源码
添加SDL播放以及OpenSLES，ANativeWindow播放本地视频，网络流，解码音频文件，
包含音视频同步等功能，rtmp推流功能和拉流功能
遗留问题：推流过程伴随有噪声，播放效果体验不佳，后期会补充webrtc，speex降噪处理
